NOTE

- Quando viene avviato il server, la directory deve avere il formato "path/" 
  (ad esempio ./files/).
- Quando il client richiede un file, deve essere richiesto il path dove si vuole 
  salvare, oltre al nome locale del file 
  (ad esempio, !get test.txt ./nomelocale.txt nel caso si voglia salvare
  nella directory corrente)
- Quando il client vuole settare la modalita' binaria o testuale, dopo averla 
  specificata deve premere invio senza aggiungere ulteriori spazi.
